## 1. 框架概述

### 1.1 项目定位

Darkit Gin 是基于 `gin-gonic/gin` 的**企业级 Web 框架增强版**，在保持 Gin 高性能的同时，提供开箱即用的企业级功能：

```
┌─────────────────────────────────────────────────┐
│            应用层 (Your Business Logic)          │
├─────────────────────────────────────────────────┤
│  增强层: JWT | SSE | Cache | OpenAPI | ZipFS   │
├─────────────────────────────────────────────────┤
│           基础层: gin-gonic/gin v1.11           │
├─────────────────────────────────────────────────┤
│              Go 1.24+ Runtime                   │
└─────────────────────────────────────────────────┘
```

### 1.2 核心特性

| 类别         | 特性          | 描述                            |
| ------------ | ------------- | ------------------------------- |
| **路由增强** | 选项式配置    | 链式调用，简洁优雅              |
|              | CRUD 快捷方法 | `router.CRUD("users", handler)` |
|              | 路由冲突检测  | 智能识别路由冲突                |

|              | OAuth 2.0     | access_token + refresh_token    |

### 2.2 Hello World

**传统方式**（兼容原生 Gin）：

```go
package main

import "github.com/darkit/gin"

func main() {
    r := gin.Default()

    r.GET("/ping", func(c *gin.Context) {
        c.Success("pong")
    })

    r.Run(":8080")

```

**推荐方式**（选项式配置）：

```go
package main

import (
    "time"
    "github.com/darkit/gin"
    "github.com/darkit/gin/cache"
)

func main() {
    router := gin.NewRouter(
        gin.WithGinMode("release"),
        gin.WithJWT("your-secret-key-min-32-chars!"),
        gin.WithCache(&cache.Config{
            TTL:             30 * time.Minute,
            CleanupInterval: 5 * time.Minute,
        }),
        gin.WithCORS("http://localhost:3000"),
        gin.WithRateLimit(100), // 100 req/min
        gin.WithRequestID(),
    )

    // 健康检查
    router.Health()

    // 业务路由
    router.GET("/api/users", listUsers)
    router.POST("/api/users", createUser)

    router.Run(":8080")
}

func listUsers(c *gin.Context) {
    c.Success(gin.H{"users": []string{"Alice", "Bob"}})
}

func createUser(c *gin.Context) {
    c.Created(gin.H{"id": 1, "name": "Charlie"})
}
```

## 3. 核心概念

### 3.1 Router vs Engine

```go
// gin.Engine - 原生 Gin 引擎
engine := gin.New()
engine.GET("/path", handler)

// gin.Router - Darkit Gin 增强路由器
router := gin.NewRouter()
router.GET("/path", handler) // 自动增强 Context
```

**关键区别**：

- `Router` 自动注入增强 Context
- `Router` 提供选项式配置
- `Router` 内置组件管理（缓存、SSE、JWT）

### 3.2 Context 对象池

**为什么需要对象池？**

```
无池化场景:
请求1 → 创建Context → GC
请求2 → 创建Context → GC  ❌ 内存分配频繁
请求3 → 创建Context → GC

有池化场景:
请求1 → 从池获取Context → 归还池
请求2 → 从池获取Context → 归还池  ✅ 减少 GC 压力
请求3 → 从池获取Context → 归还池
```

**实现机制**：

```go
// context.go:30-78
var contextPool = sync.Pool{
    New: func() interface{} {
        return &Context{}
    },
}

// 获取增强 Context
func newContext(c *gin.Context) *Context {
    ctx := contextPool.Get().(*Context)
    ctx.Context = c
    return ctx
}

// 释放回池
func releaseContext(ctx *Context) {
    // 清理引用，避免内存泄漏
    ctx.cache = nil
    ctx.sseHub = nil
    ctx.jwtAdapter = nil
    contextPool.Put(ctx)
}
```

### 3.3 配置优先级

```
环境变量 (最高)
    ↓
RouterOption (WithJWT, WithCache 等)
    ↓
DefaultConfig (框架默认值)
```

示例：

```bash
# 环境变量
export JWT_SECRET="prod-secret-key"
export GIN_MODE="release"
```

```go
// 代码配置
router := gin.NewRouter(
    gin.WithJWT(os.Getenv("JWT_SECRET")), // 读取环境变量
    gin.WithConfig(func(c *gin.Config) {
        c.ErrorHandlerEnabled = true
    }),
)
```

## 4. 路由系统

### 4.1 基础路由

```go
// HTTP 方法
router.GET("/users", listUsers)
router.POST("/users", createUser)
router.PUT("/users/:id", updateUser)
router.DELETE("/users/:id", deleteUser)
router.PATCH("/users/:id", patchUser)

// 路径参数
router.GET("/posts/:id/comments/:cid", func(c *gin.Context) {
    postID := c.Param("id")
    commentID := c.Param("cid")
    c.Success(gin.H{"post": postID, "comment": commentID})
})

// 查询参数
router.GET("/search", func(c *gin.Context) {
    query := c.Query("q")
    page := c.DefaultQuery("page", "1")
    c.Success(gin.H{"query": query, "page": page})
})
```

### 4.2 路由组

```go
// 创建路由组
api := router.Group("/api")
{
    // /api/users
    api.GET("/users", listUsers)
    // /api/products
    api.GET("/products", listProducts)

// 嵌套路由组
v1 := router.Group("/api/v1")
{
    users := v1.Group("/users")

        users.GET("", listUsers)      // /api/v1/users
        users.GET("/:id", getUser)    // /api/v1/users/:id

// 路由组中间件

    admin.GET("/stats", getStats)
```

### 4.3 CRUD 快捷方法

**方式 1：ResourceHandler 接口**

```go
type UserResource struct{}

func (u *UserResource) Index(c *gin.Context)  { /* 列表 */ }
```

### 4.4 路由冲突检测

**智能冲突检测**：

```go
router.GET("/users/:id", handler1)
router.GET("/users/:uid", handler2)  // ❌ 冲突！参数名不同但模式相同

// 日志输出:
// [GIN-ROUTER] 路由冲突检测 'GET /users/:uid' 与现有路由 'GET /users/:id' 冲突，忽略注册
```

**优先级规则**：

```go
router.GET("/users/me", handler1)       // 优先级: 1 (静态)
router.GET("/users/:id", handler2)      // 优先级: 2 (参数)
router.GET("/users/*filepath", handler3) // 优先级: 3 (通配符)

// 请求 GET /users/me    → handler1
// 请求 GET /users/123   → handler2
// 请求 GET /users/a/b/c → handler3
```

### 5.4 自定义中间件模板

```go
// 日志中间件
func LoggerMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        start := time.Now()
        path := c.Request.URL.Path

        c.Next()

        latency := time.Since(start)
        status := c.Writer.Status()

        log.Printf("[%d] %s %s - %v", status, c.Method(), path, latency)
    }
}

// 认证中间件
func AuthMiddleware(c *gin.Context) {
    token := c.GetHeader("Authorization")
    if token == "" {
        c.Unauthorized("缺少认证令牌")
        return
    }

    // 验证令牌...
    c.Set("user_id", "123")
    c.Next()
}

// 错误恢复中间件
func RecoveryMiddleware() gin.HandlerFunc {
    return func(c *gin.Context) {
        defer func() {
            if err := recover(); err != nil {
                log.Printf("Panic: %v", err)
                c.ServerError("服务器内部错误")
            }
        }()
        c.Next()
    }
}
```

## 7. JWT 认证

### 7.1 快速配置

```go
router := gin.NewRouter(
    gin.WithJWT("your-secret-key-min-32-characters"),
)
```

### 7.2 用户登录

```go
router.POST("/auth/login", func(c *gin.Context) {
    var loginForm struct {
        Username string `json:"username" binding:"required"`
        Password string `json:"password" binding:"required"`
    }

    if !c.BindJSON(&loginForm) {
        return
    }

    // 验证用户凭据
    user := authenticateUser(loginForm.Username, loginForm.Password)
    if user == nil {
        c.Unauthorized("用户名或密码错误")
        return
    }

    // 创建 JWT payload
    payload := gin.H{
        "user_id":  user.ID,
        "username": user.Username,
        "role":     user.Role,
    }

    // 生成令牌
    token, err := c.CreateJWTSession("secret-key", 2*time.Hour, payload)
    if err != nil {
        c.ServerError("生成令牌失败")
        return
    }

    c.Success(gin.H{
        "token":      token,
        "expires_in": 7200, // 秒
    })
})
```

### 7.3 保护路由

```go
// 方式 1：手动中间件
func AuthMiddleware(c *gin.Context) {
    jwt, ok := c.RequireJWT()
    if !ok {
        return // 已自动返回 401
    }

    c.Set("user_id", jwt["user_id"])
    c.Set("username", jwt["username"])
    c.Next()
}

    protected.GET("/profile", getProfile)
}

// 方式 2：内置中间件

// 方式 3：角色保护

admin.Use(router.RequireAuth(), router.RequireRoles("admin"))
{
    admin.GET("/stats", getStats)
}
```

### 7.6 OAuth 2.0 套件

```go
router.OAuth() // 自动创建 OAuth 端点

// POST /oauth/token       - 获取令牌
// POST /oauth/refresh     - 刷新令牌
// GET  /oauth/userinfo    - 用户信息
// POST /oauth/revoke      - 撤销令牌
```

**生成令牌**：

```go

func getUser(c *gin.Context, userID string) (*User, error) {
    // L1: 内存缓存
    if user, found := c.GetCache().Get("user:" + userID); found {
        return user.(*User), nil
    }

    // L2: Redis 缓存
    if user, err := getFromRedis("user:" + userID); err == nil {
        c.GetCache().Set("user:" + userID, user, 5*time.Minute)
        return user, nil
    }

    // L3: 数据库
    user, err := getUserFromDB(userID)
    if err != nil {
        return nil, err
    }

    // 回写缓存
    setToRedis("user:"+userID, user, 30*time.Minute)
    c.GetCache().Set("user:"+userID, user, 5*time.Minute)

    return user, nil
}
```

```go
            log.Fatalf("服务器启动失败: %v", err)
        }
    }()

    // 等待中断信号
    quit := make(chan os.Signal, 1)
    signal.Notify(quit, syscall.SIGINT, syscall.SIGTERM)
    <-quit
    log.Println("正在关闭服务器...")

    // 5秒超时优雅关闭
    ctx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
    defer cancel()

    if err := srv.Shutdown(ctx); err != nil {
        log.Fatal("服务器强制关闭:", err)
    }

    log.Println("服务器已退出")
}
```
